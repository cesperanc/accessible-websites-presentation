Hiperligações JavaScript

  • Hiperligações com iteração devem sempre ter o atributo href definido.
  • Isto é importante para garantir que as mesmas podem ser ativadas através de outros métodos que não apenas o clique ou o toque.
<a id="close" title="Fechar elemento">Fechar</a><!-- Errado! --> <a href="#" id="close" title="Fechar elemento">Fechar</a><!-- OK -->
