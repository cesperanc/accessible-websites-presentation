Formal structure of a Web page

A Web page is made by a document type specification, a header and a body.

<!DOCTYPE html> <html> <head> <title>Page title</title> </head> <body> Page content </body> </html>

Formal structure of a Web page (2)

  • The doctype defines the type and version of the specification which should be used to process the document.
  • In the page header (enclosed by the head tag) usually we can find the meta-data that characterize the page, as well the reference to other resources required for page layout and functionality (for instance, CSS files).
  • In the page body (enclosed by the body tag) are the actual contents of the page (like texts, images, videos, etc).
