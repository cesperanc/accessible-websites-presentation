Estrutura formal de uma página Web

Uma página Web é composta pela especificação do tipo de documento, um cabeçalho e um corpo.

<!DOCTYPE html> <html> <head> <title>Título da página</title> </head> <body> Conteúdo da página </body> </html>

Estrutura formal de uma página Web (2)

  • O doctype define o tipo e versão da especificação que deve ser utilizada para interpretar o documento.
  • No cabeçalho da página (englobado pela tag head) geralmente encontramos metadados que caraterizam a página, bem como a especificação de recursos necessários para o correto funcionamento e apresentação da página (por exemplo, ficheiros CSS e/ou JavaScript).
  • No corpo da página (englobado pela tag body) estão os conteúdos propriamente ditos (texto, imagens, vídeos, etc).
